# Rails Advanced Routing - Evening Course

## Parks and Plants

### User Story

- [ ] As a user I can see one garden's plants
- [ ] As a user I can add a plant in a garden
- [ ] As a user I can delete a plant

### Steps

1. Seed Data
2. Generate Model
3. Routes
4. Generate Controller
5. View
